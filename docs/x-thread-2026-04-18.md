我升級了 AI 記憶系統，同時解決了 10 個痛點

以前我的 AI 總是這樣：
❌ Vector Search 只知「相關」，不知為何相關
❌ 中英混合搜尋失效，中文英文當然係兩種語言
❌ 被動等你問，不會主動提你
❌ 新舊資訊不連接，舊聞同新知各不相干
❌ 矛盾資訊无人發現，之前講的同現在講的矛盾了也不知道
❌ 舊資訊一直殘留，沒有自動清理，時間久了記憶越來越雜
❌ 不記得自己知多少，亂答問題，誤導決策

升級之後（10 功能）：

✅ Triple-Stream Search — BM25 精準關鍵詞匹配 + Vector 語義理解 + Graph 關係發現，三元融合比任何單一搜尋方法都強

✅ Knowledge Graph — 每個 Entity（人物、項目、系統）都有關係標籤，不再只知「相關」，而是知道「點解相關」

✅ Smart Recall Hook — 當你提到核心關鍵詞（項目名、人名、工作相關），AI 自動觸發搜尋，不需要你每次手動要求

✅ Proactive Discovery — AI 不是等問題，而是主動發現新關係，例如：發現你正在做新項目，自動更新你的 Project List

✅ Uncertainty Tracking — 每個記憶都有 Confidence Score，AI 知道自己知幾多、不知幾多，不會亂答不確定的問題

✅ Content Half-lives — 事實類記憶有 30 天半衰期、意見類 7 天、新聞類 3 天，舊資訊自動衰減，永遠保持最新狀態

✅ Contradiction Detection — 當新資訊與舊資訊矛盾，系統主動標記舊資訊為「已過時」，避免使用過時資訊做決策

✅ Consolidation Loop — 每日記憶自動蒸餾：從對話緩衝區 → 語義 Wiki → 技能記憶，三層結構確保重要資訊進步沉澱

✅ HTTP API — 提供 REST API 接口，任何工具（TG Bot、其他 Script）都可以調用搜尋功能，不局限於 OpenClaw 內部使用

✅ Graph Visualization — 將你的人脈網絡、項目關係、系統架構全部視覺化，Debug 一目了然，解釋給別人也更容易
